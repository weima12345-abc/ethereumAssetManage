   企业信息资产管理系统的开发目的是使企业信息资产管理的模式从手工方式转变成信息管理,为企业管理提供方便条件。对企业信息资产管理的实际情况进行调研之后,进行详细的需求分析，对现有的管理模式进行改进，开发出一套新型的管理系统，从中领悟系统开发的思想，掌握系统开发的流程和方法.随着企业资产信息系统的不断完善，企业的管理将越来越依赖于信息化的管理。所以系统开发将围绕企业信息资源管理工作的实际情况,使之能迅速适应企业的需要。
 一。模块设计
（1）员工信息管理：企业管理员跟员工根据用户名与密码完成登陆.
（2）客户管理模块：企业管理员进入系统对本公司员工的资产(如名下有几台电脑，在负责什么业务等）等进行了解与掌握；对员工的资产的采购，领取和归还进行批准；
（3）系统管理模块：
1. 企业管理员进入系统对员工资产进行查询和统计。
2. 以及对此次记录对资产进行入库出库管理（记录在智能合约里）
   如何对资产流程的准确性和可追溯性：可以在合约中创建一个唯一标识，一个16位的散列值。
(4) 用户个人模块：
1.管理员： 
   查询到采购，领取和归还记录
  个人资产情况
2. 用户：
查询到采购，领取和归还记录
个人资产情况
